Simple Employability Skills Log Web Page

Features:
- Upload and post images/videos with a description
- Display all posts
- Add feedback to each post

How to use:
1. Open index.html in your browser.
2. Use the form to upload an image or video and add a description.
3. See your post appear below, and add feedback to any post.

Note: This is a static demo. Uploaded files and feedback are stored in browser memory and will reset if you reload the page.
